    Slack
    How do I get this URL?
    1. Click Get Webhook URL above (opens api.slack.com/apps).
    2. Create New App → From scratch. Name it anything (e.g. "Listener AI") and pick your workspace.
    3. Sidebar → Features → Incoming Webhooks. Toggle Activate Incoming Webhooks on.
    4. Scroll down → Add New Webhook to Workspace, pick the channel, and Allow.
    5. Copy the URL (starts with https://hooks.slack.com/services/) and paste it above.
    6. Click Send Test Message to confirm.
    If your workspace blocks app installs, ask an admin to do steps 1–4 for you.
